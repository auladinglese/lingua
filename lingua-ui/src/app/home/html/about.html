About Lingua

This is a Baltic Talents Academy Advanced JavaScript course project by Dalia Saltenyte.

Code can be found on GitHub.

Lingua is a foreign language teacher's tool for an online customised tasks assignment, evaluation and appointment management.

Available features:
  • Registration and login with a role of a teacher or a student.
  • Teacher dashboard with lessons sorted by status; upcoming appointments; pending new student and appointment requests and option to record availability.
  • Student dashboard with option to choose teacher and send appointment requests.
  • Every student has a lessons page with a lessons lists, sorted by status. Lessons can contain multiple tasks, that consist of one source (url or text) and multiple questions. Tasks are completed and saved individually. Once all tasks are completed, lesson can be submitted for evaluation.
  • Teachers can view their student's lessons page, where they can create and assign a new lesson and evaluate completed lessons.
  • Tasks for a lesson can be created using saved templates. Every task can be saved as a new template. Teachers can access and manage all of their templates (filter and full CRUD) on a separate page.
